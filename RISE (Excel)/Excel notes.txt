--8.conditional statement:
(1.)IF-THEN-   key concept: =if(condition, value_if_true, value_if_false)

=if(A2>10,"greater than 10","less than 1o")
=if(b4>=50."pass","fail")


(2.)NESTED IF-   you can include multiple logical tests within a single formula.

-example:
*marks column B
=if(B2>=90,"A",if(B2>=75,"B",if(B2>=60,"C",if(B2>=50,"D","F"))))


(3.)IF with Text-

-sales department get 15% bonus, IT get 20%, all other get 10%
=if(b2="sales",15,if(b2="IT",20,10))


(4.)IF with AND and OR-  allow you to include multiple logical test at once.   =if(logical, value_if_true, value_if_false)

-example with AND:
eligibility criteria: attendance>=75% AND exam score>=70
=if(AND(B2>=75,C2>=70),"eligible", "not eligible")


-example with OR:
scholarship rule: sports>=90 OR academic>=90 (either one qualifies)
=if(OR(B2>=90,C2>=90),"qualify", "not qualify")



(5.)IF with NOT (<>)-  if you want to evaluate a case where a logical statement is not true, you can use either the NOT statement or a "<>"operator

-example:
where exam score is not 45 print "zero", otherwise print result 
=if(NOT(C2=45),0,45)


(6.)IFERROR -  the IFERROR statement is an excellent tool to eliminate annoying error messages (#N/A, #DIV/0! , #REF! , etc.), which is particularly useful for front-end formatting.

=IFERROR(value, value_if_error)

-example:
in number table insert text
=IFERROR(B2+C2,0)


* Excel offers a number of different IS formulas, each of which check whether a certain condition is true:

-ISBLANK: checks whether the reference cell or value is blank      =ISBLANK(B2)
-ISNUMBER: checks whether the reference cell or value is numerical 
-ISTEXT: checks whether the reference cell or value is a text string
-ISERROR: checks whether the reference cell or value returns an error
-ISEVEN: checks whether the reference cell or value is even
-ISODD: checks whether the reference cell or value is odd
-ISLOGICAL: checks whether the reference cell or value is a logical operator
-ISFORMULA: checks whether the reference cell or value is a formula

-String function:

1.left  (returns 2 characters, starting from the left)     =LEFT(text, [num_chars])     =LEFT(A4,3)
2.mid   (returns 5 characters from the middle of the string, starting with position)     =MID(text, start_num, num_chars)     =MID(A4,5,4) 
3.right (returns 3 characters starting from the right)     =RIGHT(text, [num_chars])     =RIGHT(A4,3)
4.search (not case-sensitive) (returns the number of the character at which specific character or text string is first found)  =SEARCH(find_text, within_text, [start_num]) 
   =SEARCH("Pro",A2)   =IFERROR(SEARCH("Pro",A2),"Not Found")    =IF(ISNUMBER(SEARCH("Pro",A2)), "Yes", "No")     =IFERROR(IF(SEARCH("Pro",A2),"yes"),"No")

5.find  (case-sensitive)  =FIND(find_text, within_text, [start_num])   =FIND("@",A2)-1        =LEFT(A4,FIND("@",A4)-1)      =MID(A4,FIND("@",A4)+1,50)
6.len  (for finding a length)  =LEN(A2)     =IF(LEN(A2)>=3,"valid","invalid")
7.substitute (for changing text or word in string)   =SUBSTITUTE(Text, Old_text, New_text, [instance_num])  for change name  =SUBSTITUTE(A2,B2,C3)    
  for remove space  =SUBSTITUTE(A12," ","")     for remove hyphens =SUBSTITUE(B12,"-","")     for remove plus =SUBSTITUTE(C12,"+","")
  all together   =SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(A12,"-","")," ",""),"+","")
