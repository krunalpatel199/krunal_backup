1.Sorting in Excel:
- it's like you want department wise data from colleges table.

-many types of sorting in excel
(1)using Ribbon in Excel (always try to sort using select cell in column so your entire data will sorting)
   (if you select an entire column it will sorting the column but it doesn't affect on other column so it's give you inconsistent table.)
(2)custom Sorting in Ribbon (it is use when you want to sort more than one column)


2.Filtering in Excel: (shortcut key-"CTRL + SHIFT + L") 
-it's like cleaning your data for your view (like filter by department name "production")
-if you want to remove filter you can press shortcut key again or you can remove it using select filter in column name

3.Remove duplicates in Excel:
-it's removing duplicate values from your column
- for removing you need to go in data section in ribbon. after that you have to select remove duplicates and it remove duplicate value from your column

4.Validation in Excel: 
- it's like constraint or restriction for inserting value in column
- using Ribbon (you need to select entire column in which you want to apply validation then you go to data section and select data validation then you can apply)

-Error Types in Excel:

(1. #####- column isn't wide enough to display values
2. #NAME? - Excel does not recognize text in a formula
3. #VALUE! - Formula has the wrong type of argument
4. #DIV/0! - formula is dividing by zero or an empty cell
5. #REF! - Formula refers to a cell that is not valid
6. #N/A - Formula can't find a references value)
